# whatsapp-chat-analyzer-new
Whatsapp chat analyzer is a data analysis project that extracts meaningful insights from exported whatsapp chat files. it processes raw data to analyze user activity , messages, emoji usage and pattern. this tool help users understand communication behavior through interactive visualization such as timelines, activity heatmaps  and user statistics.
try analyzing your own exported whatsapp chat on - https://whatsapp-chat-analyzer-new-project.streamlit.app/
